This is a lab for NCKU C++ lesson about acquainting how to use Github and series of commands.

For compiling, you may use following command:
	g++ 3n+1.cpp
	
As for executing, the file will be named as "a.out" if you use the command above,
In order to execute, type the following command:
	./a.out
	
Then, enter a positive integer, the 3n+1 problem series of numbers will be printed.

If you have any question or suggestion, contact me @w@


這是成大c++課程中的小實驗室這樣，用來學習如何使用Github 以及相關指令的作業
編譯檔案請執行以下指令（Linux）:
	g++ 3n+1.cpp
	
如果要執行檔的話，以上的指令會產生「a.out」的執行檔，可以用以下的指令執行
	./a.out
	
開始執行檔案之後，輸入一個正整數的數字，3n+1 問題的數列就會印在終端機上面
